#  Aplikasi Konsultasi Kesehatan Sederhana (CLI) - Golang

## Deskripsi
Aplikasi ini adalah program **forum konsultasi kesehatan sederhana berbasis Command Line Interface (CLI)** yang dibuat menggunakan **bahasa Go (Golang)**.  
Program ini memungkinkan pengguna dengan peran **pasien** dan **dokter** untuk berinteraksi dalam forum diskusi kesehatan.

Pasien dapat membuat pertanyaan terkait kesehatan, sedangkan pengguna yang telah login dapat memberikan tanggapan.  
Selain itu, aplikasi ini juga dilengkapi dengan fitur **pencarian** dan **pengurutan** pertanyaan berdasarkan tag.

---

## 🎯 Fitur
- Registrasi akun
- Login dan logout
- Melihat forum pertanyaan
- Posting pertanyaan (**khusus pasien**)
- Memberikan tanggapan
- Mencari pertanyaan berdasarkan tag
- Mengurutkan pertanyaan berdasarkan tag

---

## ⚙️ Algoritma yang Digunakan
- **Sequential Search**
- **Binary Search**
- **Insertion Sort**
- **Selection Sort**
